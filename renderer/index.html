⚡ Electron 36 — Dialog API Demo

Interactive demo of every dialog module method and option

📂 dialog.showOpenDialog() async

Returns Promise<{ canceled, filePaths, bookmarks? }>. Each button below passes a different properties value or option combination.

📂 dialog.showOpenDialogSync() sync

Returns string[] | undefined — same options as async, but blocks the main process until closed.

💾 dialog.showSaveDialog() async

Returns Promise<{ canceled, filePath, bookmark? }>. Tests all major options: filters, defaultPath, buttonLabel, properties.

💾 dialog.showSaveDialogSync() sync

Returns string (empty string if cancelled). Blocks the main process.

💬 dialog.showMessageBox() async — type option

Returns Promise<{ response, checkboxChecked }>. The type option controls the icon: none | info | error | question | warning.

💬 dialog.showMessageBox() — Advanced Options

Tests: checkboxLabel, detail, noLink, normalizeAccessKeys, defaultId, cancelId, icon (NativeImage), buttons[], textWidth, alwaysOnTop.

💬 dialog.showMessageBox() — signal: AbortSignal async

Passing an AbortSignal lets you programmatically close the dialog. The demo auto-cancels after 4 seconds via AbortController.

💬 dialog.showMessageBoxSync() sync

Returns Integer — the index of the clicked button. Blocks the main process.

🚨 dialog.showErrorBox(title, content)

Modal, synchronous error box. Safe to call before the app ready event. On Linux before ready, output goes to stderr (no GUI).

🔐 dialog.showCertificateTrustDialog() macOS / Windows

Returns Promise<void>. Options: certificate (Certificate object) + message string. Not supported on Linux — a fallback message is shown instead.
